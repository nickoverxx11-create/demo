# Interactive Blog Demo

This project is an interactive blog that explains how semantic search works through a guided, game-like experience. Instead of presenting the idea as a static article, it lets readers explore how words become coordinates, how sentences become vectors, and how search systems compare meaning through interactive visuals and small tasks.

The experience is designed as a three-chapter journey so users can learn step by step, moving from single-word embeddings to query intent and finally to cosine similarity and ranking.
